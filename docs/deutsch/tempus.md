## 第一将来时

### 构成

第一将来时由变位动词 werden 构成（框型结构）：**werden + Inf.**

> Er wird mich morgen besuchen.
> 
> Ich hoffe, dass er mich moregen besuchen wird.

| 人称        | 变位动词 werden |
| ----------- | --------------- |
| ich         | werde           |
| du          | wirst           |
| er, sie, es | wird            |
| wir         | werden          |
| ihr         | werdet          |
| sie, Sie    | werden          |
