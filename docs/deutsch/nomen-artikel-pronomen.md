## 名词

德语名词可分为三类[^1]：

- **强变化名词**：大部分阳性名词，全部中性名词。
- **弱变化名词**：一部分阳性名词
    - 大部分表示人和生物的名词
    - 所有以 e 结尾的阳性名词
- **阴性名词**


### 强变化名词


[^1]: 现代德语语法

- 单数
    - 第二格 -es 或 -s
- 复数
    - 第三格 -n 或 -en，如以 -s 结尾，则无需再加词尾

### 弱变化名词

- 单数
  - 第二格 -en，以元音结尾 -n
  - 第三格 -en，以元音结尾 -n
  - 第四格 -en，以元音结尾 -n
- 复数
  - 全部格 -n 或 -en

### 阴性名词

- 单数
    - 均不变格
- 复数
    - 第三格 -n
