## 名词

### 总体变格规律

### 强变化名词

**强变化名词：**大部分阳性名词，全部中性名词[^1]。

- 单数
    - 第二格 -es 或 -s
- 复数
    - 第三格 -n 或 -en，如以 -n, -en 或 -s 结尾，则无需再加词尾

### 弱变化名词

**弱变化名词：**一部分阳性名词，其中大部分是表示人和生物的名词。

### 阴性名词

- 单数均不变格
- 复数
    - 第三格 -n，如以 -n 或 -en 结尾，则无需再加词尾
