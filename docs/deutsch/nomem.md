## 名词

### 强变化名词

强变化名词在变化时，单数第二格需加词尾 -es 或 -s，复数第三格需加 -n 或 -en（如名词的复数是以 -n, -en 或 -s 结尾的话，则不需要再加词尾），其他格都无需变化。[^1]
